# Aero — справка

**Aero** — быстрый, минималистичный и приватный браузер на движке WebKit.

## Основное
- **Адресная строка** — введите запрос или адрес. Поисковик меняется в Настройках (Google, DuckDuckGo, Яндекс, Bing).
- **Вкладки** — кнопка с числом внизу; смахните вверх по нижней панели, чтобы открыть новую вкладку.
- **Закладки и история** — в меню «···».
- **Pull-to-refresh** — потяните страницу вниз для обновления.
- **Поделиться** — через меню «···».

## Блокировка рекламы
Aero использует полноценные фильтр-листы (EasyList + EasyPrivacy + RU AdList + список против раздражителей) — сотни тысяч правил, уровень AdGuard/uBlock. Запросы к рекламным и следящим доменам блокируются на сетевом уровне WebKit.
- Включается в Настройки → Блокировка рекламы.
- Каждый список можно включать/выключать отдельно.
- **«Пожаловаться на рекламу»**: меню «···» → коснитесь ненужного блока → он скрывается навсегда.

## Импорт и экспорт
Настройки → Перенос данных.
- **Экспорт закладок** — файл HTML, открывается в Safari, Chrome и любом браузере.
- **Экспорт вкладок** — список ссылок (TXT).
- **Резервная копия** — всё в одном файле JSON; возвращается обратно через «Импорт из файла».
- **Импорт из файла** — HTML-закладки (включая экспорт Safari/iCloud), JSON-копию Aero или список ссылок TXT.
- **Импорт из буфера** — ссылки из буфера обмена.

> **Важно про Safari.** iOS не разрешает сторонним браузерам напрямую читать вкладки, куки и историю Safari — это ограничение песочницы, такого API не существует ни у одного браузера. Поэтому перенос делается через файл: в Safari «Экспорт закладок» → откройте файл в Aero. Куки перенести нельзя — на новых сайтах нужно войти заново.

## Приватность / анти-отпечаток
Настройки → Приватность. Каждый параметр включается отдельно, у каждого есть «?» с пояснением:
- Шум Canvas / WebGL / Audio
- Маскировка экрана и системных данных
- Геолокация: запрет или подмена
- Блокировка WebRTC (но реальный IP меняется только через VPN)
- Подмена языка и часового пояса

## Инкогнито
Меню «···» → Инкогнито. Отдельное хранилище, история не сохраняется, защита включается автоматически.
