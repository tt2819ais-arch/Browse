# Aero

Быстрый, минималистичный и приватный браузер для iOS (SwiftUI + WKWebView).

## Возможности (v1.1)
- Адресная строка с умным поиском (Google / DuckDuckGo / Яндекс / Bing)
- Вкладки, закладки, история, pull-to-refresh, share
- Адаптивная тема (системная / светлая / тёмная), обои на главной
- **Блокировка рекламы и трекеров** (сетевая через WKContentRuleList + косметическая)
- **«Пожаловаться на рекламу»** — коснитесь ненужного блока, и он скрывается навсегда
- **Режим инкогнито** (отдельное хранилище, без истории, защиты включены)
- **Приватность / анти-отпечаток** — granular-тумблеры: Canvas/WebGL/Audio, экран, система, гео, WebRTC, язык, часовой пояс. У каждого — кнопка «?» с объяснением
- Версия для ПК, очистка данных

## Сборка
Проект генерируется XcodeGen из `project.yml`. Сборка unsigned `.ipa` — через GitHub Actions.

> GitHub App не может пушить файлы в `.github/workflows/`. Содержимое workflow лежит в `ci/ios-ipa.yml.txt` — создайте из него `.github/workflows/ios-ipa.yml` через веб-интерфейс GitHub. Workflow сам определяет проект и схему, так что при будущих переименованиях его менять не нужно.

Артефакт `Aero-ipa-unsigned` содержит `Aero-unsigned.ipa` (unsigned). Установка: AltStore / Sideloadly / Esign.
